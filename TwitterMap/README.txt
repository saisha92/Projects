* Program to take in a Stream of Input from Twitter and Displaying the location of the Tweets in a Heat Map*

Tweetconnection - This gets the tweets and uploads it into the Amazon RDS

DBConenction - This establishes a Connection to Amazon RDS

HeatMap.jsp - This displays the Heatmap onto the browser

TEchnologies Used : JSP,Servlet,Java,Amazon Elastic Bean Stalk,Amazon RDS,Twitter4J API and Google Maps
