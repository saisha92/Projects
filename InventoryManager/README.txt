The Inventory Manager module of the Inventory Store Web Application.

The Module is a Specialized module which allows the Mobile Store manager to keep Track of the Products which is available in Store. He would be able to add, Delete and Modify the Products in the store.
 
The Manager adds and deletes the product depending on the previous purchases of the customers, the stock for the product is added based on the purchases ; If the product is popular,the number of products added to the store is large. If the product is not so popular in the market , the stock is less for the particular product. The main function of the inventory store manager is to make sure the stock is available for all the products in the store.

Used : J2EE , JSP , Servlet , HTMl , CSS
       LOG4J - To keep track of the logs and errors
